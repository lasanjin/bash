## Description
Outputs Chalmers
- Expressen
- Kårrestaurangen
- Linsen
- S.M.A.K
  
lunch menu in terminal & highlights *Köttbullar* 


## How to run
Alt 1

```
$ python expressen.py $1
```
Alt 2

1. Make script executable
```
$ sudo chmod +x ./expressen.py
```

2. Run script
```
$ ./expressen.py $1
```

- `$1`
  -  *optional* 
  -  number of days from today, default is today's menus
     -  input `0-9`
